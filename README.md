## Что это и зачем

Простое, лёгкое и безумно удобное приложение для тех, кто живёт на nolvoprosov.ru с телефона.  

## Возможности

- Полный функционал сайта: вопросы → ответы → комментарии → лайки ❤️   
- Светлая / тёмная / авто тема  
- Все категории, поиск, «Лучшее», «Горячее»  
- Профиль, рейтинг, личные сообщения  
- Никаких трекеров, аналитики и прочей фигни  
- Размер APK < 180 МБ Сжатый .zip < 80 МБ 
- Работает с Android 5.0 и выше??
